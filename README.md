# Andes Retail Dashboard (Power BI)

This project analyzes the commercial performance of **Andes Retail Group** during **2024–2025** using Power BI.

## Business Questions
The dashboard was designed to answer the following questions:

- How has total revenue evolved between 2024 and 2025?
- Which customer segments generate the highest revenue and profitability?
- Which product categories have the greatest impact on the business?
- Are there significant differences between countries or regions?
- Are there seasonal patterns affecting sales performance?

## Tools Used
- Power BI
- Power Query
- Data Visualization

## Dataset
The dataset contains transactional retail data including:

- Order ID
- Customer ID
- Product Category
- Units Sold
- Price per Unit
- Revenue
- Cost
- Country and Region
- Order Date
- Season

## Project Output
The final result is an interactive Power BI dashboard that provides:

- Executive overview of sales performance
- Country and segment comparison
- Product category analysis
- Seasonal trends in revenue
